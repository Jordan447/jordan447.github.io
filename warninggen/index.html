How to Use

1
Download the image

Right click the preview image above and select "Save image as..." to save it to your computer.

2
Upload to image host

Go to IMGbb and upload the downloaded image.

Any image hosting service that provides direct .png links will work.

3
Purchase blank notes

Go to any 24/7 store in-game and buy some blank note items.

4
Create the warning note

Use the following commands in-game:

/note [id] 1 "Traffic Warning - [Driver Name]"

Replace [id] with the blank note item ID in your inventory

/note [id] 1 "[Image URL ending with .png]"

Replace [id] with the new note item ID and [Image URL] with your uploaded image link

[!] If you keep the name and content the same for every note, they can be stacked.

5
Distribute warnings

Use the following command to give out your traffic warning notes to other players:

/giveitem [player-id] [item-id] 1

Replace [player-id] with the target player's ID and [item-id] with the note item ID

Tip: For easier management, name all your traffic warning notes consistently, e.g., "LSPD Traffic Warning" or "LSSD Traffic Warning"
